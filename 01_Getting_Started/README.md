## Getting started with Rust

1. Compile the code
```
rustc main.rs
```

2. Run the code
```
./main
```

3. Format the code layout
```
rustfmt main.rs
```

4. Starting a Cargo
```
cargo new hello_cargo
cd hello_cargo
```

5. Building the Cargo
```
build cargo 
```

6. Run the Cargo
```
run cargo
```
